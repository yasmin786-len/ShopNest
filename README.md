# 🛒 ShopNest – Full Stack E-Commerce Platform

ShopNest is a production-ready full-stack e-commerce application built with **Spring Boot 3**, **Java 21**, **React 18**, **Vite**, and **MySQL**. It provides a secure and scalable online shopping experience with modern architecture and enterprise-level best practices.

## 🚀 Features

- 🔐 JWT Authentication & Role-Based Access (Admin/Customer)
- 🛍️ Product Catalog with Search, Filter & Sort
- 🛒 Shopping Cart & Wishlist
- 💳 Secure Checkout with Mock Payment Gateway
- 📦 Order Management & Tracking
- 👨‍💼 Admin Dashboard
- 📖 Swagger API Documentation
- 🔒 Spring Security & Global Exception Handling
- 📱 Responsive User Interface

## 🛠️ Tech Stack

**Frontend**
- React 18
- Vite
- Axios
- CSS

**Backend**
- Java 21
- Spring Boot 3
- Spring Security
- Spring Data JPA
- JWT

**Database**
- MySQL

## ▶️ Run Locally

### Backend
```bash
cd ecommerce-backend
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd ecommerce-frontend
npm install
npm run dev
```
